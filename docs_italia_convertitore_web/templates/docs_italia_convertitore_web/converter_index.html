Intro

Questo strumento permette di convertire un documento esistente nel formato RST adatto a Docs Italia. Esso è tuttora in fase di test e ogni feedback sul suo utilizzo è apprezzato!

Puoi contattarci sul canale #docs-italia di Slack oppure aprire una issue su GitHub.

Modello da utilizzare

Ti suggeriamo di utilizzare uno di questi modelli preimpostati per una compilazione agevole del documento, oltre ad una conversione ottimale dello stesso

Scarica il modello (.docx) Scarica il modello (.odt)

Carica un documento

Puoi caricare file di tipo .docx, .odt, LaTeX o Markdown, creati secondo le Linee Guida su come scrivere documenti da convertire. Ti sarà inviata una e-mail all'indirizzo fornito, con lo zip da caricare su GitHub contenete files nel formato adatto a Docs Italia.
